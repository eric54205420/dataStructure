##介绍
代码为本书一些数据结构定义和算法的实现，函数列表同样来自书上的ADT定义。使用c编写，所有代码均在win7 x64上测试，编译器为gcc 4.4.1<br>
ps:dataStructure-old/目录下的内容为《数据结构》（清华大学出版社，严蔚敏）书上adt的实现，两者内容有重复，但dataStructure-old不再更新
